Detailed Description

Initializes the MPI, PETSc and SLEPc environments.

This class should be instantiated at the beginning of the program and at the end of execution the environment will close automatically Refer to Section 3 of the manuscript in /docs. For the particular case of the Ring communication approach, the basis elements are fully distributed among all available processing elements.
